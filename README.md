## Questions
- What is the total sales and total number of vehicles sold?
- Which automobile category (Car/Bike) generates the highest sales?
- How do sales vary by region, occupation, education, and commute distance?
- Which subcategories contribute most to revenue?
- What trends can be observed in customer purchasing behavior?
## Process
-  Imported raw automobile sales data into Excel
- Removed duplicates, handled missing values, and standardized data
-  Created calculated fields and structured data for analysis
- Built pivot tables to summarize key metrics
- Designed interactive visuals using charts, slicers, and KPIs
-  Add macro button to clear all slicer.
-  save file in macro-enabled mode(xlsm).
